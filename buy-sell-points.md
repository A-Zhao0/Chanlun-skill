# 三类买卖点

缠论定理：任何级别的所有买卖点，必然属于且仅属于以下三类之一（或其重合）。买点与卖点完全对称，以下以买点为主叙述。

## 第一类买点（一买）

- **定义**：某级别**下跌趋势**（至少两个中枢）中，最后一个中枢之后的趋势背驰点。
- **性质**：该级别下跌的终结点，理论上的最低点。风险：只有背驰确认后才成立；非背驰下的"低点"不是一买（这正是"当前低点尚不构成真正底部"的含义）。
- **确认**：背驰四要素齐备 + 次级别转折结构（次级别底分型/线段终结）。
- **对称**：第一类卖点 = 上涨趋势最后中枢后的趋势背驰点。

## 第二类买点（二买）

- **定义**：一买之后的次级别回抽（第一次次级别回调）的低点，**不创新低**（低于一买则一买失效）。
- **性质**：右侧确认性质，安全性高于一买，价格通常略差。
- **强化信号**：二买若伴随大阳线强势拉升，是"小转大"的典型引擎——可将本级别"新低必然"的预期直接推翻（见 divergence.md 反证法第 1 条）。
- **对称**：第二类卖点 = 一卖后次级别反抽不创新高的高点。

## 第三类买点（三买）

- **定义**：某级别中枢形成后，一次次级别走势**向上离开**中枢，随后的次级别回抽**低点不回到中枢区间**（低点 > ZG）⇒ 该回抽低点为三买。
- **性质**：中枢确认被上抛，往往开启中枢新生（趋势延续/展开）。
- **失效**：回抽重回中枢区间 ⇒ 三买不成立，中枢延伸或扩展。
- **对称（实战高频）**：**第三类卖点** = 中枢形成后次级别向下离开，回抽高点**不回到中枢区间**（高点 < ZD）。**非背驰下跌后的反弹，标准应对就是等第三类卖点**——它与"新低结构必然"的预期严格自洽：三卖成立即宣告中枢被下抛、下跌段展开。

## 位置关系与策略映射

| 当前结构状态 | 优先关注 | 逻辑 |
|---|---|---|
| 下跌趋势 + 背驰 | 一买 → 二买 | 转折启动 |
| 下跌非背驰后的反弹 | 三卖 | 新低必然，反弹是中继 |
| 中枢震荡中 | 中枢上下沿高抛低吸（次级别买卖点） | 走势未选择方向 |
| 向上离开中枢后的回抽 | 三买 | 趋势延续确认 |
| 一买/二买之间、一卖/二卖之间 | 等待，不新开仓 | 结构未确认 |

## 目标位的表述规范

- 回踩/反弹目标位来自结构测算（前中枢区间、缺口位置、次级别中枢），表述为"看至 X 附近""关注 X~Y 区间"；
- 多标的联动（如 BTC 与 ETH）可给对应目标位，但注明各自独立按结构确认；
- 永远附失效条件。
