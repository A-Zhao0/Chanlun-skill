# 缠论分析 Skill

用缠中说禅理论（《教你炒股票》108课体系）对走势做**几何结构分析**，核心不是预测涨跌，而是：

1. 把走势唯一地分解为结构（分型→笔→线段→中枢→走势类型）；
2. 对当前位置做**完全分类**，穷举后续演化路径；
3. 给每条路径挂上**确认信号与应对标准**，让操作变成"信号触发→执行"，而非猜测。

一句话概括方法论：**"不预测，只应对；不猜顶底，只做分类。"**

## 方法论立场（哲学基础）

缠论不止是画线工具，它是**哲学思想与数学工具的深度结合**，理解这个立场才能用对它——否则容易退化成"看着像要涨/跌"的主观猜测。分析时始终守住四条：

- **无常观**：市场涨跌如季节更替，本无常态。不预设方向，不因为"涨多了"就猜顶、"跌久了"就抄底。
- **当下观**：只专注"当下走势正在发生什么"，不为过去的判断和情绪买单。缠论要回答的是两个根本问题——**"你在哪"（当前结构位置）**与**"市场正在发生什么"（当前正在完成哪个级别的什么走势）**。
- **破执**：破除主观判断的执着。结论必须能被走势证伪；一旦失效信号出现，立刻放弃原判断，不扛、不找理由。
- **无我**：超越"我希望它涨/跌"的我执倾向。"多数交易者更期待上涨"这种心理本身就是要被超越的对象。

配套的操作心法："**走势必完美**"（任何级别的走势类型终要走完，未完成的结构不能提前当完成交易）与"**知者自知，不必追究也无须追究**"（新低/新高是这次来还是下次来不重要，届时按标准应对即可）。核心方法是**抽象推演优先于机械应对**：先在结构上把演化路径穷举清楚（走势分类＋反证法），而不是等现象出现了才被动反应。

数学映射（帮助理解，不必在输出里堆砌）：递归函数的结构分析（级别递归）、欧氏几何的形态定义（分型/笔/线段/中枢）、动能强弱的对比逻辑（背驰/特征序列缺口），整体与测度论相通。

## 分析工作流

拿到一张K线图或一段行情描述后，按以下顺序执行。顺序不可颠倒——缠论是递归体系，高级别结构由低级别结构定义，跳步会导致划分错误。

### 第 0 步：确认输入与级别

- 明确图表周期（1分钟/5分钟/30分钟/4小时/日线…）。图上周期 ≠ 走势级别，但它是递归的观察起点。
- 若用户上传图片：仔细读图，提取关键高低点价格、中枢箱体范围、明显的段划分（图上常有画好的笔/段折线和中枢方框，优先尊重并校验作者的划分，指出分歧处）。
- 若信息不足以定级别（例如只有一张孤图），明确说明假设："以图示周期为当下操作级别"。

### 第 1 步：结构分解（自下而上）

1. **K线包含处理**：先做包含合并，方向由前趋势决定。
2. **分型**：标出顶分型、底分型。
3. **笔**：顶底分型之间满足独立K线要求成笔。
4. **线段**：至少三笔；用**特征序列**判断线段是否终结（第一种情况：无缺口，特征序列出现顶/底分型即终结；第二种情况：有缺口，需反向特征序列分型确认）。特征序列缺口是"动能尺"——缺口不补说明反向动能弱、原趋势健康；缺口被回补是动能衰减的首个警示。
5. **中枢**：至少三个连续次级别走势类型的重叠区间，ZG = min(高点)，ZD = max(低点)。标明中枢区间、震荡次数、是否有扩展/扩张/新生。
6. **走势类型**：盘整（一个中枢）还是趋势（同向≥两个不重叠中枢）。

### 第 2 步：力度与背驰判断

读 `references/divergence.md`。核心结论必须明确二选一并给出依据：

- **背驰**：转折条件成立，进入买卖点判断；
- **非背驰**：下跌非背驰 ⇒ 结构上新低必然存在（只是不确定本轮还是下一轮实现）；上涨非背驰 ⇒ 新高必然。此时反弹/回调只做反向的第三类买卖点，不抄底/不摸顶。

背驰判断的辅助工具：MACD 黄白线与柱面积对比、同级别两段的斜率与幅度、特征序列缺口是否回补、次级别是否出现盘整背驰（区间套）。

### 第 3 步：走势分类与反证法推演

对当前位置做**完全分类**——这是缠论区别于一般技术分析的关键。标准句式：

> 后续演化路径有二（或三）：要么 A（条件、确认信号、对应买卖点），要么 B（条件、确认信号、对应买卖点）。

反证法用法：先陈述结构上的"必然"（如"非背驰 ⇒ 新低必然存在"），再列出能**推翻/推迟**该必然的全部条件（如：次级别出现第二类买点并引发小转大、伴随大阳线强势拉升；或走势通过九段盘整结构化解非背驰）。逐条核对当前是否满足。全部不满足 ⇒ 原判断维持；某条满足 ⇒ 切换到对应分类的应对。

### 第 4 步：给出应对标准（而非预测）

- 每条路径给出：触发信号、对应的买卖点类别、参考目标位/关注位、失效条件。
- 强调"知者自知，不必追究也无须追究"——新低是这次来还是下次来不重要，届时按标准应对即可。
- 目标位表述用"看至 X 附近""关注 X 区间"，不用"必到 X"。

### 输出

按 `references/analysis-template.md` 的模板组织。语言风格：技术精确、直接、不含糊；结论先行；每个判断给结构依据；允许引用缠论原文式表述（"走势必完美""即缠非缠"），但每个术语第一次出现时如上下文需要可加一句白话解释。

## 硬性约束

- **划分唯一性**：笔和线段的划分在规则下是唯一的（结合律不适用于线段划分——必须从头依序划）。若图上作者划分与规则冲突，指出来并说明规则版划法。
- **级别不混淆**：说"背驰"必须带级别（"30分钟级别背驰"），说"买点"必须带级别（"4小时第二类买点"）。无级别的判断视为未完成。
- **不做预测式断言**：禁止"必涨/必跌到X"。结构必然性（如"新低必然存在"）必须同时给出推翻条件。
- **不给个性化投资建议**：本 skill 输出的是走势结构分析与分类应对框架，不是针对用户仓位的操作指令。涉及实际下单决策时，提醒用户结合自身仓位、风险承受能力自行决策，Claude 不是持牌投资顾问。
- **图表证据优先**：分析必须锚定图上可见的价格/结构，不凭空编造点位。图上读不清的数字，标注"约"或询问用户。
