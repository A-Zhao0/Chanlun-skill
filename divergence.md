# 背驰、非背驰与转折判断

背驰是缠论中**唯一**的趋势转折充分信号。"没有趋势，没有背驰"——背驰的严格定义依赖趋势的存在。

## 1. 趋势背驰（标准背驰）

前提：同级别至少两个同向中枢构成趋势。比较对象：**最后一个中枢前后的两段同向次级别走势**（进入段 a 与离开段 c）。

c 段相对 a 段力度衰竭 ⇒ 背驰。力度比较手段（按可靠性排序）：

1. **MACD 辅助**：c 段对应的黄白线峰值与柱面积明显小于 a 段（同侧比较）；黄白线回抽零轴后的再度发散不及前次；
2. **斜率与幅度**：c 段更缓、更短，或耗时更长而幅度更小；
3. **特征序列缺口**：a 段一路留缺口而 c 段缺口被回补，佐证动能衰减；
4. **次级别结构**：c 段内部出现次级别盘整背驰（见区间套）。

## 2. 非背驰及其结构推论（实战核心）

若离开段 c 力度**不弱于** a ⇒ 非背驰。结构推论：

- **下跌非背驰 ⇒ 走势尚未完成，更低点在结构上必然存在**。当前低点不构成真正底部，任何反弹都是中继性质。
- 对称地，上涨非背驰 ⇒ 新高必然。

**策略含义**：非背驰下跌后的反弹，优先考虑（做空视角的）**第三类卖点**，与"日线新低必然"的结构预期一致；不做左侧抄底。

### 反证法：什么条件能推翻"新低必然"

"必然"是结构性的，但有明确的化解/推迟路径，必须逐条核对：

1. **次级别第二类买点引发"小转大"**，伴随大阳线强势拉升——次级别反弹强到直接改变本级别结构；
2. **九段盘整化解**：走势通过中枢延伸至9段，原级别盘整升级为更大级别中枢，非背驰状态被大级别盘整消化，新低预期随之解除或转移到更大框架内。

核对结论只有两种：
- 两条均不满足 ⇒ 新低预期维持，只是"这次来还是下次来"不确定（"知者自知，不必追究也无须追究"），按第三类卖点应对；
- 任一条满足 ⇒ 切换分类，按新结构应对。

这就是**基于走势分类的反证法**：先立结构必然，再穷举推翻条件，用当下走势逐条证伪。

## 3. 盘整背驰

只有一个中枢（盘整）时，进入段与离开段的力度对比若衰竭，称**盘整背驰**。注意：

- 盘整背驰**不保证**趋势级转折，只保证一次该级别的回拉（至少回到中枢区间）；
- 但盘整背驰是"小转大"的种子，也是区间套定位的工具。

## 4. 区间套：精确定位转折点

大级别背驰段内，找次级别的背驰段；再入更次级别……层层缩小区间，直至最低可观察级别。转折点必然落在层层嵌套的区间交集内。实战中通常做到图表可辨的两层即可（如4小时背驰段内找30分钟盘整背驰）。

## 5. 小转大

次级别（甚至更低级别）的盘整背驰/第二类买点，引发超出该级别的转折，直接改变本级别结构。识别特征：

- 转折伴随力度异常（大阳线/大阴线、放量、跳空）；
- 本级别原有结构预期（如"新低必然"）被次级别力量提前终止。

小转大是"结构必然"的主要例外来源，因此任何必然性判断都必须附带"除非次级别小转大"的但书。

## 6. 判断输出规范

任何背驰结论必须写全四要素：

> **[级别] + [比较对象 a/c 两段] + [依据（MACD/幅度/缺口/次级别结构）] + [结论：背驰 / 非背驰 / 盘整背驰]**

例："30分钟级别，第二中枢离开段相比进入段 MACD 柱面积缩减约一半且缺口被回补，判定趋势背驰。"
