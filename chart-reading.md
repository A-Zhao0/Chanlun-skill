# 实盘标注截图解读

用户常上传**已带缠论标注的行情截图**（TradingView / 富途牛牛 / Gate / Binance 图）。解读前先按本文认全图上的符号约定，再校验作者划分是否合规，最后才叠加自己的分类推演。**不要无视作者已画好的东西另起炉灶**——优先尊重、指出分歧、给出规则版依据。

## 一、买卖点编号约定（最重要）

标注图上散布的小数字，是三类买卖点的速记。通行约定：

| 标记 | 含义 | 位置特征 |
|---|---|---|
| **1** | 第一类买点（一买） | 下跌趋势末端、背驰低点 |
| **2** | 第二类买点（二买） | 一买后次级别回抽不破前低处 |
| **3** | 第三类买点（三买） | 中枢上方、离开后回抽不回中枢 |
| **-1** | 第一类卖点（一卖） | 上涨趋势末端、背驰高点 |
| **-2** | 第二类卖点（二卖） | 一卖后次级别反抽不创新高处 |
| **-3** | 第三类卖点（三卖） | 中枢下方、离开后反抽不回中枢 |

要点：
- **负号=卖点方向，正号=买点方向**；数字大小=第几类，不是级别。
- 颜色辅助：绿色常标在买方确认/底部转折处，红色常标在卖方/顶部转折处，但**以数字和位置为准，颜色只是提示**。
- 同一张图会**多级别混标**（宏观 4H 的大 1/-2 ＋ 局部的小 1/2/3）。判断每个编号属于哪个级别，看它锚在哪一段折线/哪个中枢上。
- 一个位置可能**同时是不同级别的不同买卖点**（如"1分钟三买"恰在"30分钟二买"位置）——这是区间套和"小转大"的观察点，不是矛盾。

## 二、图形元素约定

- **折线（zigzag，紫/青/红实线）**＝作者划的**笔或线段**。连接相邻分型高低点。校验：每段是否满足成笔/成段要求（独立K线数、特征序列）。斜率陡的段＝力度强。
- **色块方框（粉/灰/蓝/青半透明矩形）**＝**中枢**。框的上下沿即 ZG/ZD。留意方框是否覆盖≥3段次级别走势；框被向上/向下离开且回抽不回，对应三买/三卖。
- **红色/绿色斜直线（较长、常虚线）**＝**趋势线/背驰辅助线**（作者手画，非缠论严格定义），用于直观看动能斜率衰减，仅作参考。
- **水平线群（带斐波数字，如 0.618 / 2.618 / 4.236）**＝斐波回撤/扩展位，**非缠论原生**，是作者叠加的目标位工具。可引用为"关注位"，但不作为结构判断依据。
- **副图 MACD（DIF/DEA/柱）**＝**背驰判断的主要依据**。读法见下。

## 三、副图 MACD 判背驰（配合 divergence.md）

标注图几乎都带 MACD 副图，这是把"力度对比"落地的关键：

- **同侧柱面积对比**：比较趋势中前后两段同向走势（进入段 a vs 离开段 c）对应的 MACD 柱子面积。c 段柱面积明显小于 a ⇒ 背驰。
- **DIF 距零轴**：价格创新低/新高，但 DIF 未同步创新低/新高（更靠近零轴）⇒ 顶/底背驰的典型形态。
- **黄白线（DIF/DEA）回抽零轴**：回抽后再度发散的力度不及前次 ⇒ 动能衰减。
- 若图上无 MACD 副图，**明确告知力度判断无法完成，背驰/非背驰结论悬置**，并请用户补 MACD 图——这是硬缺口，不要用形态硬猜。

## 四、未来路径投影图的读法

有的图（如纯投影图）会用**虚线框＋折线画出尚未走出的未来路径**：

- **黑色/白色虚线方框**＝作者预判**正在构建或即将构建的中枢**（未来中枢的预期区间）。
- **延伸出的折线＋末端箭头**＝**走势分类推演的可视化**：画出"构建中枢→向某方向离开→是否创新低/新高"的预期路线，箭头指向预判的目标（如指向新低）。
- 解读时说清：这是**一种分类下的推演路径，不是必然**。对应到反证法——把这条路径成立需要的条件、以及能推翻它的条件（小转大/九段盘整）一并列出。可以在自己的分析里复用这种"画出未来中枢框＋投影折线"的表达，但永远附确认信号与失效条件。

## 五、解读流程（拿到标注图后）

1. 认周期与标的（表头），认清是**不复权/前复权**（不复权时历史价位不可直接跨期比较）。
2. 认全买卖点编号，判断每个属于哪个级别。
3. 校验折线（笔/段）与色块（中枢）是否合规，指出与规则的分歧。
4. 读副图 MACD，落地背驰/非背驰判断；无 MACD 则悬置并请求补图。
5. 若图上有投影路径，转述其分类含义并补上反证条件。
6. 叠加自己的完全分类与应对标准（回到主 SKILL.md 工作流第 3、4 步）。

## 注意：表头 OHLC 陷阱

TradingView/富途 表头显示的 开/高/低/收 常是**十字光标停留处那根K线**的数据，不是最新价。最新价看右侧价格轴的高亮标签或行情区报价。读图时先分清，别把光标处的历史K线当成当前价（这是高频误读点）。
