# 缠论核心概念（递归定义链）

整个体系是一条递归链：**K线 → 分型 → 笔 → 线段 → (最低级别中枢) → 走势类型 → 更高级别中枢 → …**。每一层由下一层严格定义，因此分析必须自下而上。

## 1. K线包含处理

两根K线若一根的高低点完全包含另一根，需合并：

- 前趋势向上：合并后高点取两者较高，低点取两者较高（向上包含处理）；
- 前趋势向下：高点取两者较低，低点取两者较低。

包含处理后才谈分型。

## 2. 分型

- **顶分型**：三根（处理后）K线，中间K线的高点最高、低点也最高。
- **底分型**：中间K线的低点最低、高点也最低。

分型是转折的最小单元，但单个分型不构成转折确认——强分型（带长影线/放量）意义大于弱分型。

## 3. 笔

一顶一底之间构成一笔，要求：

- 顶分型与底分型之间至少有1根不属于两分型的独立K线（即老笔标准下顶底分型加中间至少共5根处理后K线）；
- 顶必须高于底（向上笔），或反之；
- 相邻分型必须是顶底交替；连续两个顶分型取更高者，连续两个底分型取更低者。

## 4. 线段与特征序列（重点）

**线段**：至少由三笔构成，且前三笔必须有重叠区间。

### 特征序列

设向上线段由笔序列 S1 X1 S2 X2 … Sn Xn 构成（S=向上笔，X=向下笔）：

- 任何 Si 与 Si+1 之间必有重合区间（否则线段早已终结），所以 S 序列携带的信息少；
- **X1 X2 … Xn（逆向笔序列）才真正刻画线段的性质**，称为该向上线段的**特征序列**。向下线段的特征序列则是其向上笔序列。

把特征序列的每个元素当作一根"K线"（取其高低点区间），可以做包含处理、可以定义顶分型/底分型——这就是判断线段终结的工具。

### 特征序列缺口与线段终结的两种情况

**缺口**：特征序列相邻两元素之间没有重合区间。

- **第一种情况（无缺口）**：向上线段中，特征序列出现顶分型，且分型的第一、二元素间**无缺口** ⇒ 线段在该顶分型的高点处终结，无需进一步确认。
- **第二种情况（有缺口）**：特征序列顶分型的第一、二元素间**有缺口** ⇒ 不能直接确认终结。必须等待：从该分型高点开始的反向走势中，其（反向的）特征序列也出现底分型，才确认原线段终结；若反向走势不能形成确认（缺口方向再创新高/新低），原线段延续。

### 缺口作为"动能尺 / 情绪尺"（实战解读，重点）

把特征序列缺口理解为一把衡量市场内在动能强弱的**"情绪尺"**。它解决一个非常实际的问题：当前这段走势是真的**"力竭而亡"**了，还是只是**"中场休息"**、后续还会继续发力？两条用法：

- **识别强势趋势**：上涨趋势中，每次回调形成的特征序列都出现缺口 ⇒ 下跌动能非常弱，当前上涨趋势非常健康，**不要因为涨多了就轻易猜顶做空**。（下跌趋势对称：每次反弹特征序列都留缺口 ⇒ 上涨动能弱、下跌健康，别急着抄底。）
- **警惕趋势转折**：之前一直有缺口，但某一次回调终于把缺口**回补**了（回调力度加大，与前一元素产生重叠）⇒ 重要警示信号，表明原有强劲动能可能正在衰减，需开始密切关注趋势是否可能转变，配合背驰判断确认。

核心作用一句话：**客观衡量市场内在动能的强弱，判断当前走势的"健康程度"与"延续可能性"，避免被表面价格波动迷惑**。缺口是否回补，是背驰之外一个独立且更早的动能信号，实战中与 MACD 柱面积、斜率对比互相印证（见 `divergence.md`）。

## 5. 中枢

**定义**：某级别走势类型中，被至少三个连续**次级别**走势类型所重叠的部分。

- 记三段次级别走势的高低点，则中枢区间 **[ZD, ZG]**，其中 ZG = min(三段高点)，ZD = max(三段低点)；
- 中枢区间的高低延伸：ZG之上的最高触及为GG，ZD之下最低触及为DD。

中枢的三种复杂形态：

- **中枢延伸**：围绕同一中枢的次级别震荡不断增加（不突破 [ZD,ZG] 的有效范围），最多延伸到9段——**9段完成即构成更大级别中枢/盘整**（这正是"九段盘整化解非背驰"的结构基础）。
- **中枢新生**：离开段走出后新的三段重叠不与原中枢重叠 ⇒ 同向新中枢，构成趋势。
- **中枢扩展/扩张**：两个同级别中枢震荡区间产生接触/重叠，升级为更大级别中枢。

## 6. 走势类型与级别递归

- **盘整**：该级别内只包含**一个**中枢。
- **趋势**：包含**至少两个**同向排列且互不重叠的中枢（上涨/下跌）。
- **级别递归**：最低级别（如1分钟）的线段充当"次级别走势类型"生成1分钟中枢→1分钟走势类型；1分钟走势类型作为次级别生成5分钟中枢……依此类推到30分钟、日线、周线。所以"4小时级别中枢"的严格含义是：由至少三段30分钟级别（或图表可辨的次级别）走势重叠构成。

**走势必完美**：任何级别的任何走势类型终要完成。一个走势类型一旦确认，它必然走到出现该级别的背驰/终结结构为止；未完成的结构（如中枢只有两段重叠）不能提前当作完成来交易。

## 7. 同级别分解

把走势严格按同一级别切分为"上涨+盘整+下跌"的连接序列，不允许跨级别混用。作用：

- 消除"级别选择的任意性"带来的判断漂移；
- 在同级别分解视角下，"当前正在构建X级别中枢"与"新低结构上必然存在"这类判断才有严格含义——因为盘整之后接续的方向由背驰与否唯一约束。
